GET /api/album/:name -> Buscar un album
POST /api/album -> Crear un album
DELETE /api/album/:name -> Borrar un album
PATCH /api/album/:name -> Modificar un album

POST /api/artista/ -> Crear  artista
DELETE /api/artista/:name -> Borrar artista
PATCH /api/artista/:name -> Modificar artista

Explicaciones:
- Node: Entorno para ejecutar JavaScript en el servidor, fuera del navegador.
- Npm: Gestor de paquetes para instalar y administrar librerías de JavaScript.
- Express: Framework minimalista para crear servidores web y APIs de forma rápida.

Node.js te permite correr JavaScript → npm gestiona paquetes → Express simplifica crear aplicaciones web.
